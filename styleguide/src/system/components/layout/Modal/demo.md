## Basic Modal

Basic modal usage

```html
<ds-modal style="position: absolute" />
```

```
<template>
  <div>
    <ds-modal 
      v-model="isOpen"
      title="Modal Title" 
    >
      <p>Hello World</p>
    </ds-modal>
    <ds-button primary icon="rocket" @click="isOpen = true">Open Modal</ds-button>
  </div>
</template>
<script>
  export default {
    data() {
      return {
        isOpen: false
      }
    }
  }
</script>
```
